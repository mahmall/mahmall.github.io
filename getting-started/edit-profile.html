# تعديل الملف الشخصي

يمكنك تعديل ملفك الشخصي بعد تأكيد هاتفك عن طريق كود التحقق وتنقسم البيانات التي يمكنك تعديلها إلى الآتي :

  • بيانات عامة
  • بيانات تعتمد على نوع الحساب

# بيانات عامة

البيانات العامة هي البيانات التي تشترك فيها كل الحسابات بكل أنواعها وبمختلف خصائصها ومميزاتها وهي كالآتي :

  • الاسم
  • البريد الإلكتروني
  • رقم الهاتف
  • كلمات المرور
  • الصورة الشخصية
  • تحديد عنوانك بشكل تلقائي

# بيانات تعتمد على نوع الحساب

تلك البيانات التي تختلف باختلاف نوع الحساب سواء حساب فردي أو حساب متجر وهي كالآتي :

  • كتابة العنوان بشكل تقريبي (للحسابات الفردية)
  • كتابة عنوان متجرك (للحسابات التجارية)
  • اسم متجرك (للحسابات التجارية)
  • اسم صاحب المتجر (للحسابات التجارية)

يُرجي الانتباه إلى أن تغيير رقم الهاتف يتطلب إعادة تأكيده عن طريق كود التحقق
